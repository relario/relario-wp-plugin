Micropayments solution to monetize traffic from hard-to-reach countries.

== Description ==

Relario PAY offers easy to use integration for customers to provide their users an easy way to donate to them using our micropayment solution.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/relario-pay` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Relario Pay screen to configure the plugin
1. Use the Short code on the pages that you need the plugin.
